⚠️ VIKTIGT - CORS Problem:

Webbläsare blockerar direkta API-anrop (CORS-policy). För produktion behöver du en backend-server. Ollama fungerar bäst för lokala test!

💡 Rekommenderade lösningar:

  • 🦙 Ollama (Bästa för test): Ladda ned Ollama
  • 🔧 Backend proxy: Skapa Node.js/Python server
  • 🌐 CORS Proxy: cors-anywhere
  • 📱 Browser Extension: Installera CORS-disabled extension

🔗 API-nycklar (för när du har backend):

  • 🤖 OpenAI API Key
  • 🧠 Claude API Key
  • 💎 Gemini API Key
  • ⚡ Groq API Key
